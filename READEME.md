# Chess Opening Trainer
## Frontend Build Setup
1. Install dependencies 
```
npm install
```
## Backend Dev Setup
1. The psychopg2 postgresql connector library is built from source, therefore you must fulfill the build requirements: https://www.psycopg.org/docs/install.html#prerequisites
2. Once the requirements are met, dependencies can be installed from requirements.txt
```
$ cd backend
$ pip install -r requirements.txt
```
